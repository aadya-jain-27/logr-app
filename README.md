# Logr

**Your goal, broken into today.** Logr is a calm, AI powered study and career planner. It turns a long term goal into a small, doable plan for today, adapts as life happens, and lays the whole path across a calendar so you can trust it fits your deadline.

## Features

- **Onboarding** that captures your goal, real available time, study resources (courses, YouTube links, PDFs/PPTs), and fixed commitments.
- **Today**: two to four focused tasks sized to your time, generated by Google Gemini, with a "where to start" tip.
- **The Path**: your goal broken into phases with projects to build and a nudge to push them to GitHub.
- **Calendar** (week and month): spreads your resources across real dates, honors "divide across N weeks", and marks your exams.
- **Adaptive planning**: carries over unfinished work, welcomes you back gently after a gap, and keeps commitment days lighter.
- **Add your own tasks** day to day, a **Pomodoro** focus timer, **month at a glance** progress, and **six immersive worlds** with procedural ambient sound.

## Tech

- React 19, Vite, React Router, Tailwind CSS v4, Framer Motion
- Google Gemini API, called **only server side**
- Serverless API routes (`api/*`) in production; a Vite dev middleware mirrors them locally
- Browser storage for your profile, plan, and progress (privacy first, no account needed)

## Getting started

```bash
npm install
cp .env.example .env      # then add your free Gemini key
npm run dev               # http://localhost:5180
```

Get a free Gemini key at https://aistudio.google.com/apikey

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint

## Project structure

```
api/            Serverless functions (plan, roadmap, parse-file, parse-url)
src/
  screens/      Full pages (Onboarding, Today, Path, Calendar, Journey, Settings, ShareCard)
  components/   Reusable UI (TopBar, FocusTimer)
  scenes/       The six animated worlds and shared scene parts
  lib/          gemini (prompts + API), plan, ambient, notify, util
  data/         store (localStorage: profile, plan, roadmap, extras, history)
  theme / focus / sound   React context providers
vite.config.js  Vite config + dev-only API middleware
vercel.json     SPA rewrites for production
```

## Environment & security

`GEMINI_API_KEY` must be set (locally in `.env`, in production as a Vercel environment variable). It is read only on the server and never sent to the browser. `.env` is gitignored.

## Deployment

Deploys to Vercel: import the repo, add `GEMINI_API_KEY` as an environment variable, and deploy. `vercel.json` rewrites client routes to `index.html`, and `api/*` run as serverless functions.
